# Проект: [Место](https://hotess.github.io/mesto/)

Описание проекта
--------------------

Проект о путеществии. В нем описаны несколько живописных мест

Используемые технологии
-----------------------------------
* Адаптивная верстка;
* Флексбокс;
* Грид;
* Фигма;
* Оптимизация картинок;
* Гит;
* Разработан по методологии **БЭМ**.
