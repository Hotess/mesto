# Проект: [Место](https://hotess.github.io/mesto/src/)

Описание проекта
--------------------

Проект о личности и о его путешествии в разных живописных местах.

Используемые технологии
-----------------------------------
* Адаптивная верстка;
* Флексбокс;
* Грид;
* Фигма;
* Оптимизация картинок;
* Гит;
* Разработан по методологии **БЭМ**.
